## Getting Started

1. Install Python 3.9+.
2. Install Jupyter:
   ```bash
   pip install notebook
   ```
3. Start Jupyter Notebook in this folder:
   ```bash
   jupyter notebook
   ```
4. Open any assignment notebook and run cells from top to bottom.

## Notes

- Some notebook filenames include spaces and mixed capitalization.
- If you are opening notebooks from the terminal, wrap filenames in quotes.
- It is recommended to create and use a virtual environment for dependencies.

## Suggested Workflow

- Read the assignment prompt in the first markdown cells.
- Run starter cells first.
- Implement solutions incrementally and re-run tests/output cells.
- Save and checkpoint your notebook frequently.
